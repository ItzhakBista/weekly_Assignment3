# מטלה שבועית (שבוע 3)- משחק חלליות

## תיאור:
הוספת שיפורים למשחק חלליות שראינו בהרצאה בתוכנה Unity בעזרת Assets ושינויים בקוד.

## מטרת המטלה:
פיתוח יצירתיות ושיפור יכולת קידוד בנסוף להכרת הממשק העזרת מימוש טריגרים ודגמים.

## שיפורים שנוספו למשחק
* רקע זז
* הוספת גבולות- עולם עגול בציר האופקי ועולם סגור בציר האנכי.
* תצוגת ניקוד קבוצה בפינה הימנית למטה.
* חיים- לשחקן יש 3 נקודות חיים, הוא נהרס לאחר שנפסל 3 פעמים. יש ספנר של לבבות שהשחקן יכול לאסוף ולקבל חיים.
* מעבר שלבים- השחקן עובר שלב לאחר שצבר מספר נקודות מסוים לכל שלב.
